![xp-screenshot](https://user-images.githubusercontent.com/46838874/121505388-6946f200-ca00-11eb-8443-44271d7ed274.png)

```
Usage: winfetch [/c] [/?]

Options:
    /c             Show colors (using ANSI escape sequences).
    /?             Display this text.

Thanks to:

    SaladFingers !SpOONsgtAo
    Jz9 !//QwUWqnYY
```
